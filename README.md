# ⚡ SMART VOLT - Complete Electrical Company Management System

## 🚀 Overview
SMART VOLT is a comprehensive web-based management system designed specifically for electrical companies.

## ✨ Features

### 👨‍💼 Admin Dashboard
- Dashboard Analytics with real-time statistics
- User Management (Add, Edit, Delete employees/admins)
- Employee ID Management
- Attendance Management with filters
- Reports & Analytics (Monthly reports, absentee lists, late percentages)
- Salary Management (Auto-calculate salaries based on attendance)
- Basic Salary Settings

### 👤 Employee Dashboard
- QR Code Attendance with camera
- Manual Check In/Out option
- Profile Management with photo upload
- Attendance History
- Salary History

### 🌐 Company Website
- Modern Glassmorphism landing page
- Online Booking System with email notifications
- Contact Form (Messages to Admin + WhatsApp)
- Services Showcase

### 🤖 AI Chatbot
- OpenAI GPT-3.5/4 powered
- Human-like conversations
- Unlimited questions

## 🛠️ Tech Stack
- Laravel 12, PHP 8.2
- MySQL Database
- Bootstrap 5
- OpenAI API
- QR Code Library

## 🔧 Installation

```bash
git clone https://github.com/YOUR_USERNAME/smartvolt-attendance.git
cd smartvolt-attendance
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan storage:link
php artisan serve
